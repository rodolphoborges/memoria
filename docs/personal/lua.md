# Lua Documentation

Lua is a powerful, efficient, lightweight, embeddable scripting language. It supports procedural programming, object-oriented programming, functional programming, data-driven programming, and data description.

## Reference Manuals
The reference manual is the official definition of the Lua language.

* **[Lua 5.4 Reference Manual](https://www.lua.org/manual/5.4/)** (Current Version)
* **[Lua 5.3 Reference Manual](https://www.lua.org/manual/5.3/)**
* **[Lua 5.1 Reference Manual](https://www.lua.org/manual/5.1/)** (Legacy - widely used in game engines)

## Books
### Programming in Lua (PiL)
Written by Roberto Ierusalimschy, Lua's chief architect, it is the authoritative introduction to all aspects of Lua programming.

* **First Edition:** Available online for [free](https://www.lua.org/pil/contents.html). Covers Lua 5.0.
* **Fourth Edition:** Updated for Lua 5.3.

## Official Resources
* **[Executive Summary](https://www.lua.org/about.html):** Overview of what Lua is and why to use it.
* **[Lua Tutorial Directory](https://lua-users.org/wiki/TutorialDirectory):** Community-maintained wiki with practical examples.
* **[Lua Workshop](https://www.lua.org/wshop/):** Archives of presentations and papers from previous workshops.

## Technical Papers
* *The Evolution of Lua (2007)*: A historical view of the language's development.
* *Lua 5.0 and the Spirit of LUA (2005)*: Details on the register-based virtual machine.
